# Real-Time Chat Application

## Description
A full-stack real-time chat application built using the MERN stack. The app supports secure authentication, instant messaging, media sharing, and live user presence with a clean and modern UI.

## Features
- Custom JWT-based authentication (no third-party auth).
- Real-time one-to-one messaging using Socket.io.
- Online/Offline user presence indicators.
- Typing indicators with sound notifications (toggle supported).
- Image sharing with Cloudinary integration.
- Secure RESTful APIs built with Node.js & Express.
- MongoDB for reliable data persistence.
- API rate-limiting using Arcjet for enhanced security.
- Responsive and modern UI with React, Tailwind CSS & DaisyUI.
- Global state management using Zustand.

## Tech Stack
- Frontend: React.js, Tailwind CSS, DaisyUI
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-Time: Socket.io
- Authentication: JWT
- Media Storage: Cloudinary
- State Management: Zustand
- Security: Arcjet Rate Limiting
- Tools: VS Code, Git & GitHub
